### Tailwind Config

Extend Tailwind with all CSS variables as custom colors. Map `brand`, `bg`, `panel`, `text-muted`, `warning`, `danger`, `info` to CSS variables.

### Design Rules (ENFORCE STRICTLY)

- **Glass morphism**: All cards/panels use `backdrop-filter: blur(16px)` + semi-transparent background
- **Border radius**: 12–28px on all surfaces, never sharp corners
- **Mint glow**: Active nav items, focused inputs, primary buttons have `box-shadow: 0 0 20px rgba(11, 231, 164, 0.3)`
- **Typography**: Use `Plus Jakarta Sans` (import from Google Fonts) — bold for headings, regular for body
- **Spacing**: Generous — minimum 16px padding on mobile cards
- **Dark mode**: Default. Light mode toggled via `data-theme="light"` on `<html>`
- **Every interactive element**: Has `hover:` state + `active:scale-95` with 200ms transition
- **No solid white backgrounds** — always use CSS variable surfaces

## ANIMATIONS (FRAMER MOTION SPECS)

### Page Transitions
```jsx
const pageVariants = {
  initial: { opacity: 0, x: 20 },
  animate: { opacity: 1, x: 0, transition: { duration: 0.3, ease: "easeOut" } },
  exit: { opacity: 0, x: -20, transition: { duration: 0.2 } }
}
```
Wrap every page in `<AnimatePresence>` + `<motion.div variants={pageVariants}>`.

### Card Reveals (staggered list)
```jsx
const containerVariants = {
  animate: { transition: { staggerChildren: 0.07 } }
}
const cardVariants = {
  initial: { opacity: 0, y: 16 },
  animate: { opacity: 1, y: 0, transition: { duration: 0.35 } }
}
```

### Modal Entrance
```jsx
{ initial: { opacity: 0, scale: 0.92 }, animate: { opacity: 1, scale: 1 }, exit: { opacity: 0, scale: 0.95 } }
```

### FAB
```jsx
{ whileHover: { scale: 1.08 }, whileTap: { scale: 0.94 } }
```

### XP Bar fill: `animate={{ width: `${percent}%` }}` with spring transition.

### Circular Focus Ring: SVG `stroke-dashoffset` animated with Framer Motion.

### Level Up Modal: Scale + confetti burst using `canvas-confetti`.

### 7. GPA CALCULATOR

**Grading Scales**:
- University of Ghana (UG): A=80-100 (4.0), B+=75-79 (3.5), B=70-74 (3.0), C+=65-69 (2.5), C=60-64 (2.0), D+=55-59 (1.5), D=50-54 (1.0), F=0-49 (0)
- KNUST: Mark-based percentage system
- US/Canada 4.0: A=93-100, A-=90-92, B+=87-89, B=83-86, B-=80-82, C+=77-79, C=73-76...
- UK: First=70+, Upper Second=60-69, Lower Second=50-59, Third=40-49
- Custom: User-defined ranges

**Semester Manager**: Add/remove semesters. Each semester: name + list of courses.

**Course Entry**: Name + credit hours + grade (letter or raw score input, auto-converts).

**Calculations**:
- Weighted GPA = Σ(credits × points) / Σ(credits)
- Cumulative GPA across all semesters
- Baseline: input prior GPA + credits for transfer students

**What-If Simulator**: Toggle to hypothetical mode. Adjust grades with sliders. GPA updates in real time with animated number.

**Target Tracker**: Set target GPA. Shows required average for remaining credits. Color coded: achievable (mint) / stretch (warning) / unlikely (danger).

**Grade Predictor** (math-based, no AI):
- Inputs: current GPA trend (slope from last 3 semesters) + weekly study hours
- Formula: predicted_gpa = current_gpa + (trend_slope × 0.5) + (study_hours_delta × 0.02)
- Displayed as a projected range with confidence label

**GPA Trend Chart**: Recharts line chart showing GPA per semester.

### 11. XP & ACHIEVEMENT SYSTEM

**XP Sources**:
- Focus session done: +10 XP
- Focus session partial: +5 XP
- Focus session attempted: +1 XP
- Task completed: +5 XP
- Daily streak maintained: +3 XP

**Level System**: Every 100 XP = level up. Level-up triggers full-screen modal: level number animates in + confetti burst + badge if unlocked.

**10 Achievement Badges**:
1. First Session — complete first focus session
2. Week Warrior — 7-day streak
3. Ten Hours — 10 total focus hours
4. Task Master — 50 tasks completed
5. GPA Guardian — set a GPA target
6. Night Owl — session after 10pm
7. Early Bird — session before 7am
8. Perfect Week — hit weekly target 4 weeks in a row
9. Comeback Kid — restore a streak after breaking it
10. Scholar — reach GPA 3.5+

Each badge: locked (grayscale) → unlocked (color + glow animation + toast notification).

**Streak**: Consecutive days with ≥1 completed focus session. Flame icon. Resets at midnight if no session.

## CODE QUALITY RULES

- **Zero dead code**: Every component must be rendered somewhere. Every function must be called.
- **Component size**: Max 200 lines per component. Split if larger.
- **All animations**: Via Framer Motion only. No CSS `@keyframes` for interactive animations.
- **Styling**: Tailwind utility classes only. No inline styles except Framer Motion `animate` props.
- **Mobile-first**: All Tailwind classes base = mobile. `md:` for tablet. `lg:` for desktop.
- **State**: React Context for global state (auth, theme, app data). Local `useState` for UI state only.
- **Firestore**: All reads/writes in `lib/firestore.js`. No direct Firestore calls in components.
- **Error handling**: Every async operation wrapped in try/catch. User-facing error toasts on failure.
- **Loading states**: Every data fetch shows a skeleton loader, not a spinner.
