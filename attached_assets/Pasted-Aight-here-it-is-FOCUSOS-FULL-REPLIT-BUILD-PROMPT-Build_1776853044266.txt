### Design Rules
- Dark mode default, smooth light/dark toggle accessible from anywhere in the app
- All surfaces use glass morphism — `backdrop-filter: blur()` with semi-transparent panels
- Soft corners everywhere: 12–28px border radius
- Subtle mint glow on active/focus elements
- Typography: Clean, modern sans-serif. Large confident headings, small muted labels
- Spacing: Generous padding, breathable layouts — never cramped
- Every interactive element has a hover + active state with smooth transition (200–300ms)
- Motion: Smooth, decisive, low-friction. No jarring snaps. Use Framer Motion for all page transitions, modal entrances, card reveals, and list animations
- Mobile-first: Designed for 375px+ screens, scales up to desktop beautifully

## VIEWS & FEATURES

### 1. Home (Focus Cockpit)
- Animated greeting based on time of day ("Good morning, George")
- Widget-style swipeable cards dashboard (use Framer Motion drag or react-swipeable):
  - Today's Tasks card
  - Focus Stats card (study time today, weekly streak)
  - Upcoming Exams card
  - GPA Snapshot card
- XP bar with level and progress
- Big "Start Focus Session" button — mint, glowing, prominent
- Signal Center — alerts for overdue tasks, upcoming deadlines, attendance warnings
- Quick shortcuts to all main sections
- Exam mode auto-activates when an exam is within 48 hours (visual change on home)

### 2. Plan (Task Management)
- Create tasks with: title, priority (high/medium/low), estimated time, due date, course link
- Filter by priority, course, due date
- Task cards with smooth entrance animations
- Swipe left to delete (mobile), swipe right to complete
- Undo toast notification (5 seconds)
- Start focus session directly from a task
- Schedule task into a time block

### 3. Calendar (Events & Timetable)
- Monthly/weekly calendar view toggle
- Add events: exams, deadlines, classes
- Weekly timetable grid with attendance tracking per class
- Auto-creates recovery tasks for missed classes
- Gap study suggestions between classes
- Prep task auto-generation before exams
- Events color-coded by type (exam = danger red, deadline = warning orange, class = info blue)

### 4. Schedule (Time Blocking)
- Create time blocks linked to tasks
- Conflict detection — warns if blocks overlap
- Day view and list view toggle
- Start focus session from a block
- Mark block complete or delete
- Smooth drag-to-reorder (desktop)

### 5. Analytics (Progress Dashboard)
- Completion rate, adherence rate, risk score — displayed as clean stat cards
- Charts (Recharts):
  - Tasks by priority (donut chart)
  - Weekly focus trend (area chart)
  - GPA trajectory by semester (line chart)
- Progress bars with animated fill on load
- Context warnings if metrics drop below threshold

### 6. Smart Hub (Insights)
- Context-aware study tips and recommendations
- Burnout detection based on focus session patterns
- Mood check-in (daily emoji selector)
- Weekly comparison (this week vs last week)
- Automatable actions: rescue overdue tasks, streak guard, auto-plan sessions
- All recommendations rendered as clean actionable cards

### 7. GPA Calculator
- Multi-university grading scales: UG, KNUST, US/Canada 4.0, UK Percentage, Custom
- Semester-based course entry (credit hours + grade or raw score)
- Automatic weighted GPA calculation
- Cumulative GPA across semesters
- What-if simulator — adjust hypothetical grades, see GPA update in real time
- Target GPA tracker — shows required average for remaining credits
- Grade Predictor — math-based prediction using GPA trend + logged study hours
- GPA trend line chart (Recharts)

### 8. Focus Session System
- Launch from task, block, or home
- Configure duration: 25–90 min custom + Pomodoro presets (25/5, 50/10, 90/20)
- Pomodoro Customizer — set custom work/break ratios and cycles
- Running timer with circular progress ring (animated)
- Pause/resume
- Background ambient/lo-fi music player during session (Howler.js)
- On end: quality selector (Done +10XP, Partial +5XP, Reschedule)
- Keyboard shortcuts
- Auto-restore session on reload
- beforeunload warning if session is active

### 9. Study Music Player
- Built into focus session screen
- Tracks: Lo-fi, Ambient, Rain, White Noise, Deep Focus
- Play/pause, volume control, track selector
- Minimal UI — doesn't distract from timer
- Persists across session

### 10. Notifications
- Browser push notifications for:
  - Task due reminders (1 hour before)
  - Exam countdowns (24 hours and 1 hour before)
  - Class reminders (15 minutes before)
  - Streak at risk alert
- Toggle per notification type in settings
- In-app notification bell with unread count badge

### 11. Profile & Settings
- User profile (name, university, year)
- Study preferences (session length, break length, daily goal)
- Display settings (dark/light mode, font size, high contrast)
- Course + GPA management
- Notification preferences
- Data export (JSON)
- Account: Google sync, guest upgrade, sign out

```
users/{uid}/
  profile
  tasks/{taskId}
  events/{eventId}
  blocks/{blockId}
  sessions/{sessionId}
  courses/{courseId}
  classSessions/{classId}
  moodLogs/{logId}
```
